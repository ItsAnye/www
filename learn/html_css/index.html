    • HTML, also know as Hyper Text Markup Language, is the language for the web. Most of the websites you go to are created with HTML, including Meltdown! In this course, you will learn how to create your very own website. If that sounds good to you, then let's get into it!

    HTML is made up of tags. The first tag we are going to learn is the <h1> tag:

    See where we wrote <h1>? That is called an opening tag. Notice how after we typed <h1> , we added </h1> This is called a closing tag. The main difference between an opening and closing tag is that in a closing tag, you have to put a / before the tag name. In an <h1> , the closing tag would like this: </h1> .
